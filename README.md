# Python Data Structures Assignment

This repository contains Python programs based on dictionary and list operations.

---

## Problem 1: Student Marks Using Dictionary

### Description
This program:
1. Creates a dictionary where student names are stored as keys and their marks as values.
2. Takes a student's name as input from the user.
3. Retrieves and displays the corresponding marks.
4. Displays an appropriate message if the student's name is not found.

### Concepts Used
- Dictionary
- User Input
- Conditional Statements

---

## Problem 2: List Extraction and Reversal

### Description
This program:
1. Creates a list of numbers from 1 to 10.
2. Extracts the first five elements from the list.
3. Reverses the extracted elements.
4. Prints both the extracted list and the reversed list.

### Concepts Used
- List
- List Slicing
- Reverse Operation
